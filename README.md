# Tic-tac-toe (C++17)

This repository contains source code for a classic Tic-tac-toe game which is implemented using C++17.

```
============================
Welcome to play Tic-tac-toe!
============================
 _____ _____ _____
|     |     |     |
|  1  |  2  |  3  |
|_____|_____|_____|
|     |     |     |
|  4  |  5  |  6  |
|_____|_____|_____|
|     |     |     |
|  7  |  8  |  9  |
|_____|_____|_____|

Player (X) -> Choose a slot number between 1 - 9:
```

## How to compile and run

```
make all
./bin/tic_tac_toe
```
